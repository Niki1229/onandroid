Android Networking for the Enterprise

Instructors: Blake Meike

This class seeks to turn journeyman Android developers into solid Android architects by leading them through successively more complex implementations of a network layer, examining the benefits and disadvantages of each. Predominantly lab-oriented, we will implement three different architectures, ending with the challenging and ill-documented SyncAdapter.

Course Level: Intermediate

Prerequisites:

  • A laptop with Android development tools installed.
  • Facility with the Android development tools.
  • Facility with the Java programming language.
  • While the morning session will be accessible to anyone with a little experience, the labs will get increasingly demanding.
  • Participants who have built at least one complete Android application will get the most out of this course.
  • Cost$200

  • Time 9:00am - 4:00pm
  • LocationSFSU (Room TBA)

Java Fundamentals

Instructors: Tim Lee & Nathan Esquenazi

This is an introduction to Java, designed for beginners and focused on teaching the Java concepts necessary to do Android development with Eclipse. This course is especially well-suited to people already familiar with HTML5 and/or Javascript but looking to get more hands-on experience with Java and OOP (Object Oriented Programming).

The course will review basic programming concepts such as methods, conditionals, loops, objects and classes, inheritance and interfaces, ArrayLists and HashMaps, and some basic drawing. We will be building a hands-on project during the workshop.

Course Level: Beginner

Prerequisites:

  • Some existing development experience with HTML5, Javascript, or any existing programming language / environment.
  • Windows or Mac laptop to bring to the course to complete hands-on projects relating to the concepts
  • Android Development Tools (ADT) installed on your computer. Steps here: https://www.dropbox.com/s/ifzym2sjfosvc6z/Installing%20Eclipse%20and%20Android%20SDK.pdf
  • Cost$200
  • DateMonday October 21
  • Time 9:00am - 4:00pm
  • LocationSFSU (Room TBA)

Android Development Fundamentals

Instructors: Tim Lee & Nathan Esquenazi

This is an introduction to developing apps for the Android platform. The course will cover the required developer tools, app fundamentals like activities and intents, layout managers, basic controls like buttons and lists, styling, menus and dialogs, supporting multiple screens, and publishing your app. You will develop your own Android application through the course.

Course Level: Beginner

Prerequisites:

  • Existing experience with object oriented principles and design
  • Existing experience with the Java programming language
  • Windows or Mac laptop to bring to the course to complete hands-on projects relating to the concepts
  • Android Development Tools (ADT) installed on your computer. Steps here: https://www.dropbox.com/s/ifzym2sjfosvc6z/Installing%20Eclipse%20and%20Android%20SDK.pdf
  • Cost$200
  • DateFriday October 25
  • Time 9:00am - 4:00pm
